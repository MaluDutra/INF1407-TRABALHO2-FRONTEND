## Como Executar o Projeto Localmente

1. Instalar dependências:
   ```bash
   npm install
   ```
2. Compilar o TypeScript:
   ```bash
   npm run build
   ```
3. Abrir `index.html` em um servidor local (ex: extensão "Live Server" do
   VS Code, ou `python -m http.server`).
4. Garantir que o backend (Django) esteja rodando e que a `API_BASE_URL` em
   `src/api.ts` aponte para o endereço correto.

> Veja `CONTRATO-API.md` para a lista de endpoints que este frontend espera
> do backend.
